jQuery Sheet Plugin

A small (and well, opinionated) lightbox-style plugin for notifications. A trigger element (can use <a> or href attr or a data attribute on any other element to target local, in-page html and clones it into sheet to display over an overlay. A basic sheet consists of a title and any html content.

  • settings can be html data attributes or as an options object passed via the plugin instantiation
  • Small js and css files
  • Support for aria events
  • One or multiple triggers on a page – each with it's own settings.
  • Triggers can be anything on the page, including within an active sheet
  • Uses CSS Transforms for smooth animation
  • OnOpen and onClose callbacks
  • tiny to large screens
